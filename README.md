# 🔐 ASP.NET Core 2FA Implementation using OTP.NET

This project demonstrates how to implement **Two-Factor Authentication (2FA)** in an **ASP.NET Core Web API** using the **OTP.NET** library.

The application uses **Time-based One-Time Password (TOTP)** similar to authenticator applications such as:

- Google Authenticator
- Microsoft Authenticator
- Authy

Users log in using **username and password**, and then verify their identity using a **6-digit OTP generated on their mobile device**.

---

# 🔑 How 2FA Works

1. User registers in the system.
2. The server generates a **secret key**.
3. The secret key is stored in the database.
4. The user adds the secret key to an **Authenticator App**.
5. The authenticator app generates a **6-digit OTP every 30 seconds**.
6. During login:
   - User enters **username/password**
   - User enters **OTP**
7. The server verifies the OTP using **OTP.NET**.

# 🔌 API Endpoints

## 1️⃣ Register User

Creates a new user and generates a **2FA secret key**.

**Endpoint**


POST /api/User/Register


**Request**

```json
{
  "userName": "testuser",
  "name": "Test User",
  "password": "123456"
}

Response

User Created. secret key is XXXXX

Add this secret key to your Authenticator App.

2️⃣ Login

Validates username and password.

Endpoint

POST /api/User/Login

Request

{
  "id": 1,
  "password": "123456"
}

Response

Enter 2FA code in Login2FA method generated from phone.
3️⃣ Verify 2FA

Verifies the OTP generated by the authenticator app.

Endpoint

POST /api/User/Verify2FA

Parameters

id : User ID
otp : 6 digit OTP

Example

POST /api/User/Verify2FA?id=1&otp=123456

Response

{
  "token": "generated-token"
}
⚙ Secret Key Generation

Secret keys are generated using OTP.NET KeyGeneration.

private string Generate2FASecretKey()
{
    var bytes = KeyGeneration.GenerateRandomKey(20);
    return Base32Encoding.ToString(bytes);
}
🔍 OTP Verification Logic
var totp = new Totp(Base32Encoding.ToBytes(user.TwoFactorSecretKey));

if (!totp.VerifyTotp(otp, out _))
{
    return BadRequest("Invalid 2FA code");
}

The server verifies the OTP generated by the authenticator application.

⚠ Security Recommendations

For production systems:

Never store plain text passwords

Use password hashing (BCrypt or ASP.NET Identity)

Implement JWT authentication

Provide QR code for secret key setup

Implement backup codes

Add rate limiting for OTP verification attempts
